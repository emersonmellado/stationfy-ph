# stationfy-ph

> Project started with Yeoman generator for AngularJS + Gulp.

> * web best pratices.
> * guidelines powered by Google.

> Gulp provide fast workspace with quick feedback.


## Usage

### Run

```
git clone https://github.com/emersonmellado/stationfy-ph.git
npm install
bower install
gulp serve
```
